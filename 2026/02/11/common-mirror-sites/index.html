常用镜像网站汇总

在国内访问一些国外网站时，经常会遇到速度慢或者无法访问的问题。本文整理了一些常用的镜像网站，可以帮助我们更快速地获取资源。

npm 镜像

官方 npm 镜像

  • 官网：https://www.npmjs.com/

国内镜像源

  • 阿里云镜像：https://npmmirror.com/
  • 华为云镜像：https://mirrors.huaweicloud.com/npm/
  • 腾讯云镜像：https://mirrors.cloud.tencent.com/npm/

如何使用镜像源

GitHub 镜像

官方 GitHub

  • 官网：https://github.com/

常用镜像服务

  • GitHub代理加速：https://ghproxy.com/
  • GitHub代理（支持HTTPS）：https://ghproxy.net/
  • FastGit：https://fastgit.org/
  • 中国科学技术大学镜像：https://g.yzu.edu.cn/
  • 上海交通大学镜像：https://github.j jl.sc.cn/

国内镜像加速器

  • 华为云容器镜像服务：https://swr.cn-north-4.myhuaweicloud.com/
  • DaoCloud：https://hub.daocloud.io/
